安全可靠

基于SSH协议建立安全隧道，提供SOCKS5和HTTP代理服务，确保数据传输安全。

灵活配置

支持多种启动参数，可自定义监听地址、端口，支持多种代理模式和过滤规则。

管理界面

内置管理界面，实时查看连接状态、日志和配置信息，便于监控和调整。

快速开始

基本使用

使用以下命令启动SSH隧道：

./ssh-tunnel -s xx.xx.xx.xx

这将使用默认设置启动SSH隧道，在本地1081端口开启SOCKS5代理。

查看更多命令选项

主要功能

SOCKS5代理

默认在本地1081端口提供SOCKS5代理服务

HTTP代理

可选开启HTTP代理，默认在本地1082端口

域名过滤

支持HTTP请求域名过滤功能

HTTP Basic认证

可配置HTTP代理的基本认证，提高安全性

管理界面

内置Web管理界面，默认地址localhost:1083/view/index

系统服务支持

支持Windows服务和MacOS自启动配置

管理界面预览

SSH-Tunnel提供了直观的管理界面，方便监控隧道状态和配置应用。

  • 实时查看SSH连接状态
  • 查看应用日志
  • 查看和编辑域名过滤列表
  • 查看应用配置信息
