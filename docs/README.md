## 文档分类说明

### 功能文档 (features/)
包含各个功能模块的详细说明，包括：
- 功能概述
- 使用方法
- API接口
- 技术实现
- 故障排除

### 部署文档 (setup/)
包含系统部署、配置和维护相关的文档，包括：
- 多平台部署指南
- 服务配置说明
- 环境要求
- 安装步骤
- 运维指南

## 如何使用这些文档

1. **新用户**: 从 `index.html` 开始，然后查看 `installation.html`
2. **开发者**: 查看 `config-api.md` 和 `features/` 目录下的功能文档
3. **运维人员**: 重点关注 `setup/` 目录下的部署和配置文档
4. **问题排查**: 查看 `faq.html` 和相应功能文档的故障排除部分
